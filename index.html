FER Spotify Music Recommender

A Desktop Application that uses Facial Emotion Recognition (FER) to analyze facial expressions from static images in order to reveal information on one's emotional state. Based on the mood, it recommends various songs that the user might like based on their Spotify history. Made in Python with Pandas, Tekore for Spotify Data Extraction, and ScikitLearn for a Random Forest Classifier.

Python Pandas ScikitLearn RandomForestClassifier Tekore
Dota 2 Heroes Recommender System

A Web Application that recommends heroes to pick (from the Dota 2 Multiplayer Game) based on the current draft, in order to increase the Match Win Rate. Made in Python with Flask Web Framework, Pandas, ScikitLearn, Monte Carlo Tree Search Algorithm and LightGBM as RandomForestClassifier. Deployed on Heroku.

Python Flask Heroku LightGBM Recommender MonteCarloTreeSearch Pandas ScikitLearn
World Cups Interactive View

A Web Application to implement Raw SQL Queries. It shows the FIFA World Cup history from 1930 to 2018, including data such as Teams, Players, Awards and Matches. Made in Django Web Framework with SQL Raw Queries, and MySQL as Database.

Python MySQL Django
